Reflection & Learning Journey

Throughout this course, my development methodology evolved from building static UI components to orchestrating complex, full-stack mobile applications. Bridging Jetpack Compose with reactive state management (MVVM) was challenging but ultimately provided a highly efficient, low-maintenance codebase. Integrating hardware sensors and asynchronous network calls (Retrofit/Firebase) taught me crucial debugging and DevOps skills, specifically how to navigate API data mapping and emulator network configurations. I am proud to have built a tool that functionally addresses SDG 12.
